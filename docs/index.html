Introduction

HTML is the default choice for portably formatting text. However, the language is not particularly well-suited to be written by hand, leading to a variety of competing formats used to generate it, such as Markdown, Wiki markup or BBCode as well as more generic formats like reStructuredText or AsciiDoc.

Bs1 markup is another competitor in this space. You might want to consider it if the following constraints are compatible with your use case:

  • you're fine with generating 'tag soup' instead of well-structured markup
  • you want some control over the generated markup, but do not need to generate arbitrarily deeply nested code
  • reading TeX does not make you want to claw out your eyeballs

If that's not something you can live with, you're probably better off with another solution.
